💰 مدير فلوسي – Money Tracker

وصف المشروع:
برنامج ويب لمتابعة الدخل والمصروف الشخصي، حساب الرصيد، وتنظيم المصاريف حسب الفئة والحساب. يعمل على الكمبيوتر والموبايل، ويمكن استخدامه أوفلاين أو أونلاين مع Firebase لمزامنة البيانات بين الأجهزة.

المميزات:

🌙 Dark Mode احترافي

💸 تسجيل الدخل والمصروف مع الفئة، الحساب، والتاريخ

📝 إمكانية إضافة ملاحظات لكل عملية

📊 رسم بياني لتوزيع المصروفات حسب الفئة

📅 فلترة العمليات حسب الشهر والحساب

💻 يعمل على الكمبيوتر والموبايل

🔒 بيانات محفوظة أونلاين مع Firebase أو أوفلاين باستخدام LocalStorage

🖤 نصوص الدخل والمصروف باللون الأسود للرؤية الواضحة

طريقة الاستخدام:

فتح الملف money-tracker-final.html أو money-tracker-firebase.html على متصفحك

إضافة الدخل أو المصروف، مع اختيار الحساب، الفئة، التاريخ والملاحظة

عرض العمليات في الجدول أو رسم بياني لتوزيع المصروفات

تعديل أو حذف أي عملية بسهولة بالضغط على الأزرار ✏️ 🗑

تشغيل النسخة أونلاين (اختياري):

استخدام Firebase Realtime Database لمزامنة البيانات بين الأجهزة

ربط Firebase بالملف عبر firebaseConfig
